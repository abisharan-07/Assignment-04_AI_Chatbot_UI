# NeuraChat – AI Chatbot UI

##  Scenario Chosen
This project is a **Modern AI Chatbot UI** similar to ChatGPT, designed using  
**HTML, CSS, and JavaScript**.  
It showcases a clean login page, responsive chat interface, and modern UI components.
